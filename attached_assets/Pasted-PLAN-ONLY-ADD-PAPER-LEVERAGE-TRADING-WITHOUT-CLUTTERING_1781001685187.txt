Core formulas:
margin = user collateral
leverage = selected multiplier
position_size = margin × leverage

For long P&L:
price_move_percent = (current_price - entry_price) / entry_price
unrealized_pnl = position_size × price_move_percent
ROI_on_margin = unrealized_pnl / margin

Example:
10 SOL margin at 10x = 100 SOL exposure.
If token moves +10%, P&L = +10 SOL.
If token moves -10%, P&L = -10 SOL and position is liquidated.

Simple liquidation:
liquidation_drop_percent = 1 / leverage

Examples:
2x liquidates around -50%
5x liquidates around -20%
10x liquidates around -10%
20x liquidates around -5%

Use a small maintenance buffer only if needed, but explain it clearly.

Liquidation behavior:

* If current MC/price reaches liquidation threshold, close position automatically.
* Mark trade as Liquidated.
* User loses margin only.
* Do not allow leverage position to make account equity negative.
* Add liquidation to trade history and admin stats.
