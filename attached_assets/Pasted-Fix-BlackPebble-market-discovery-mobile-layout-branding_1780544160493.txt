PART 1 — LIVE TOKEN DISCOVERY

Audit current token sources.

Remove:

* Hardcoded token lists
* Static demo tokens
* Long-lived cached arrays
* Stale trending lists

Build a backend token discovery service using:

DexScreener:

* token-profiles/latest/v1
* token-boosts/latest/v1
* token-boosts/top/v1
* tokens/v1/solana/{addresses}

PumpPortal:

* subscribeNewToken
* subscribeMigration

Use a single websocket connection.

Create separate live feeds:

Trending:

* Recent activity
* Volume
* Liquidity
* Transactions
* Momentum

Top Gainers:

* Highest 24h gainers
* Minimum liquidity required

Volume:

* Highest 24h volume
* Solana only

New Launches:

* Live PumpPortal stream
* Hydrate with DexScreener when available

Refresh:

* Backend: every 60 seconds
* Frontend: every 30–60 seconds

Display:
“Last Updated: X seconds ago”

* Missing symbol
* Missing name
* No price
* No liquidity
* No volume
* No market cap or FDV
* No recent activity
