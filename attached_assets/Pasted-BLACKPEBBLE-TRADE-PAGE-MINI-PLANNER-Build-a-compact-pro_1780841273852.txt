BLACKPEBBLE TRADE PAGE MINI PLANNER

Build a compact professional Mini Trade Planner directly into the token trading page.

Goal:
Allow users to plan a trade while viewing a chart and immediately apply that plan to their paper trade.

IMPORTANT:
This is NOT the full Utilities Trade Planner.
The Utilities version remains the advanced calculator.
This is a lightweight execution assistant.

PLACEMENT

Desktop:

Chart

Buy/Sell Panel

Mini Trade Planner

Positions / History / Watchlist

Mobile:

Buy/Sell Panel

Mini Trade Planner

Positions / History / Watchlist

Do not place above the chart.
Do not place below positions.
Do not create a separate page.

COLLAPSIBLE DESIGN

Default state:
Collapsed

Header:

Trade Planner ▼

When collapsed:
Only show header row.

When expanded:
Show planner controls.

Remember expanded/collapsed state during the session.

MINI PLANNER INPUTS

Auto-fill current token market cap.

Fields:

Entry Market Cap
(default current MC)

Target Market Cap

Stop Market Cap

Investment Amount

Investment Unit Toggle:

SOL

USD

CALCULATIONS

Show:

Expected Position Value

Projected Profit

Projected Return %

Risk %

Reward %

Risk/Reward Ratio

Target Multiple

Example:

Risk / Reward

1 : 4.2

Expected Value

12.4 SOL

Projected Profit

+7.4 SOL

APPLY TO TRADE

Button:

Apply To Trade

When clicked:

Fill existing trade amount field.

Store target MC and stop MC as planned trade values.

Do not execute a trade.

User must still manually click Buy.

PLANNED TRADE DISPLAY

After Apply To Trade:

Show small planned trade summary:

Target MC

Stop MC

Expected Return

Risk/Reward

Visible near the buy/sell controls.

NO AUTOMATION YET

Do NOT build:

Automatic order execution

Background workers

Take profit execution

Stop loss execution

Limit orders

Database order triggers

Those will be built later.

MOBILE

Must remain compact.

No large cards.

No long forms.

No scrolling inside components.

BlackPebble styling only.
