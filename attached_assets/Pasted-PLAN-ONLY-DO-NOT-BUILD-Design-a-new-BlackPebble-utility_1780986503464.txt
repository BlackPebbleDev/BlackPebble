====================================================
CAMPAIGN CREATION

Campaign creator enters:

* Token Name
* Token Symbol
* Token Address
* Goal Amount (SOL)
* Deadline
* Campaign Description

System creates:

* Campaign ID
* Campaign Page
* Unique Deposit Address
* Funding Tracker

Each campaign must be isolated.

No shared funding wallets.

====================================================
MULTIPLE CAMPAIGNS

The platform must support:

Campaign A
Campaign B
Campaign C

all running simultaneously.

Funds must never mix.

Each campaign must have:

* Unique address
* Unique tracking
* Unique balance
* Unique contributor history
