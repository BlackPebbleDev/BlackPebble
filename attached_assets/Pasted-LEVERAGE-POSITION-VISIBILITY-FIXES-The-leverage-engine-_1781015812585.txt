LEVERAGE POSITION VISIBILITY FIXES

The leverage engine is working, but leverage visibility on the token trading page needs to be improved.

Current issue:
The token trading page shows the normal Spot position value and unrealized P&L under the buy/sell box, but leveraged positions are not shown in the same area.

Also, the existing Positions / History box only shows normal Spot positions and Spot history. Leveraged positions/history need their own separate box.

Do not break existing Spot positions.
Do not break existing Spot history.
Do not mix Spot positions and Leverage positions together.
Do not include leverage P&L in Spot leaderboard stats.

====================================================

1. ADD LEVERAGE SUMMARY UNDER BUY/SELL BOX
    ====================================================

Under the buy/sell box, where Spot currently shows:

Position value
Unrealized P&L

Add an equivalent leverage summary when the user has an open leverage position for the selected token.

If the user has a Spot position only:

* show the existing Spot summary only

If the user has a Leverage position only:

* show the Leverage summary only

If the user has both Spot and Leverage positions:

* show both summaries stacked cleanly

Recommended labels:

Spot Position

* Position value
* Unrealized P&L

Leverage Position

* Margin used
* Position size
* Current value
* Unrealized P&L
* ROI
* Liquidation MC
* Distance to liquidation

Keep it compact and mobile-friendly.

Leverage current value formula:
current_position_value = notional + unrealized_pnl

or if token exposure is stored:
current_position_value = exposure_tokens × current_price

Distance to liquidation:
For long positions:
distance_to_liq_percent = ((current_market_cap - liquidation_market_cap) / current_market_cap) × 100

Display clearly:
Distance to Liq: 4.2%

3. ACTIVE ORDERS CLARITY

Do not label open leverage positions as normal Active Orders.

However, if leverage TP/SL orders exist, show them under Active Orders as:

Leverage TP/SL

Example:
MUMU 20x Long TP — Close 100% at MC ≥ $4.5M
MUMU 20x Long SL — Close 100% at MC ≤ $3.45M

If no Spot orders and no leverage TP/SL orders exist, keep the current empty state.
